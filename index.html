Aurora Store - Online Clothing Store

×

Aurora Store is a fictional online shopping website built using React, Redux, Routing, Firebase and much more.

Features

  • Signing up and Signing in the user with Google and Email
  • Storing products in the cart in the database for syncing over all devices
  • Searching products from all the products currently available

Technologies used and Implementations

  • React for ui
  • Redux for state management
  • Firebase for authentication and storing cart data of a user
  • React router for routing between different pages

  • Quick suggestions on highlighting text
  • Bookmarking the words
  • Pronouncing the word. Dictator for reading inputted text
  • Three Theme options to choose: Light, Dark, Colorful

Technologies used and Implementation

  • Interacting with two APIs
